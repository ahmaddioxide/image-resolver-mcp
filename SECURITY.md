# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.2.x   | :white_check_mark: |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability, please report it responsibly:

1. **Do not** open a public issue.
2. Email the maintainers or open a **private** security advisory on GitHub (Security tab → Advisories → Report a vulnerability).
3. Include a description, steps to reproduce, and impact assessment if possible.
4. Allow time for a fix before disclosing publicly.

We will acknowledge receipt and work on a fix. Thank you for helping keep this project safe.
